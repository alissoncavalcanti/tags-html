HTML

Voltar

A tag principal de uma página HTML, é ela que envolve todas as outras tags da página e torna possível a construção dos blocos essenciais para sua montagem.

head

Voltar

O head também é uma das tags principais do HTML. Ela abriga a tag title, que muda o título da página, e a tag style, que faz mudanças no CSS.

body

Voltar

A tag body abriga a maioria das tags da página HTML. É nela que estarão as outras tags com imagens, textos, tabelas e muito mais que usamos para compor sites e programas.

title

Voltar

O title dá o título à página HTML. É o seu conteúdo que aparece na aba do navegador.

b

Voltar

A tag b deixa o texto em negrito, assim como a tag strong, porém possui uma relevância semântica menor, principalmente para leitores acessíveis de páginas web.

strong

Voltar

Muito parecida com a tag b, a tag strong torna o texto negrito. Comparada à tag b, a tag strong possui uma relevância semântica maior e dá mais destaque ao texto quando lido por um programa de leitura acessível de páginas web.

u

Voltar

A tag u torna o texto sublinhado. A letra que dá nome à tag é a inicial da palavra inglesa underline, ou linha inferior (sublinhado). É usada para destacar termos.

i

Voltar

A tag i torna itálicas as palavras envolvidas por ela. Este é um recurso muito utilizado para destacar palavras estrangeiras.

h1, h2, h3, h4, h5 e h6

Voltar

Estas são tags para títulos textuais, sendo h1 a maior tag e h6, a menor. O título h1, h2, h3, h4, h5 e h6, por exemplo, foi escrita com a tag h2. Para textos menores e longos constuma-se usar outra tag, a p.

p

Voltar

A tag p abre um parágrafo. É reservada para textos longos e com fonte de tamanho menor que os títulos. Aqui é interessante utilizar recursos como links, negritos, itálicos e muito mais.

blockquote

Voltar

Blockquote é uma tag que formata o texto em formato de citação, com maior margem que o texto da tag p.

ul

Voltar

Esta tag cria uma listagem não ordenada. Com ul, os itens (criados com a tag li) são precedidos de um bullet point e devem ser criados entre a abertura e o fechamento da tag ul.

ol

Voltar

A tag ol cria uma lista ordenada, com os itens iniciados por números iniciando por 1. Estes itens devem ser inseridos entre a abertura e o fechamento da tag ol, com a tag li.

li

Voltar

li é a tag que cria itens a serem inseridos em listas, sejam elas não ordenadas ou ordenadas.

a

Voltar

Uma tag muito utilizada atualmente, a é utilizada para criar links. O link é fornecido a partir do atributo href e tem o comportamento definido pelo atributo target, com as opções de abertura do endereço na mesma ou em páginas novas.

img

Voltar

Similar à tag figure, a tag img faz a inserção de uma imagem à página HTML. Ela aceita os atributos src, que traz a origem da image, height, que define a altura e width, que define a largura. Apenas informar um dos valores faz com que a imagem seja redimensionada proporcionalmente.

form

Voltar

Muitas das inserções de informações são feitas a partir de formulários. E é para isso que serve a tag form. No formulário são inseridos campos de entrada de informações como inputs.

input

Voltar

Os campos de texto para inserção de informações dentro entre tags form são criados por meio das tags input, sendo elas com o tipo definido pelo atributo type, com as opções de ser text, ou texto comum, number, ou números, button, ou botão, entre outras possibilidades.

button

Voltar

A tag button cria um botão clicável, com um dos principais atributos sendo o type, com as opções button, sendo um botão clássico, reset para limpar um formulário ou submit, para enviar os campos de um formulário.

select

Voltar

select cria uma lista expansível para seleção de opções, sendo utilizada em conjunto com as tags label para nomear o campo e form para envolver inputs e selects.

label

Voltar

A tag label exibe rótulos para campos de formulário como inputs ou selects.

textarea

Voltar

Esta tag é utilizada para fazer uma inserção de texto em um formulário, tendo mais exibição de linhas que o input.

font

Voltar

A tag font é utilizada para manipular a fonte dentro de um bloco de texto. Se formos, por exemplo, mudar a fonte de uma palavra específica, pode-se envolvê-la com a tag font e manipular seu CSS.

div

Voltar

A tag div é muito utilizada no HTML por criar blocos estruturais que abrigam diversas outras tags. Possui comportamento de block, ocupando toda a largura da página, diferente da tag span.

span

Voltar

A tag span tem comportamento semelhante à da div, mas enquanto aquela apresenta o atributo display como block, esta apresenta como inline e permite ocupar apenas a largura do seu conteúdo.
